# pharmbrew

Pharmbrew is a robust pharmaceutical management system built using Flutter for cross-platform compatibility. This application is designed to streamline and enhance the operations of pharmaceutical industries, providing an efficient solution for managing various aspects of the pharmaceutical supply chain.
